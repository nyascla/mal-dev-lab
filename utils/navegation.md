# Navegación VSCode + Vim

## Colapsar
- `Ctrl + K`, `Ctrl + 1`
- `Ctrl + K`, `Ctrl + j`

## Saltar
- `Ctrl + P` → fichero
- `Ctrl + Shift + O` → símbolos

## Vim
- `gd` → definición
- `Ctrl + O` → atrás
- `Ctrl + I` → adelante

## a PrAcTiCaR

- `*` → Buscar la palabra bajo el cursor **hacia adelante**.
- `#` → Buscar la palabra bajo el cursor **hacia atrás**.
- `u` → **Deshacer** la última acción.
- `Ctrl + r` → **Rehacer** la acción deshecha.
- `.` → Repetir la última acción.

---
- `c`: “change” = borrar + entrar en modo Inserto.
- `i`: inner -b= () y B = {}
	- `viw`: seleccionar una palabra
	- `ci"`: borrar el contenido entre "" y escribir
	- `diw` and `diW`: borrar una palabra
	- `ysiw"`
