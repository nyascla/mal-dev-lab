# Sysinternals

## Process Explorer

Visión en tiempo real de procesos y sus recursos
- Handles abiertos
- DLLs cargadas
- Procesos activos y su jerarquía

## Process Monitor (Procmon)

Sniffer de actividad de sistema, no solo procesos.
- Archivo I/O
- Registro de Windows
- Procesos y threads (creación, terminación).
