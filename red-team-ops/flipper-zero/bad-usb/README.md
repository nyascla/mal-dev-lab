# Lenguajes

- `DuckyScript` clásico (el nativo del Flipper): rápido de escribir, pero limitado y frágil con teclados no ES/EN.
- `DuckyScript 3.0` + multi-layout (recomendado 2025): soporta ALT+ASCII, cambios de layout y es mucho más fiable.
- `BADBLE` (si usas firmware custom como RogueMaster o Xtreme): permite scripts en Python directamente desde el Flipper → brutal para lógica compleja.
